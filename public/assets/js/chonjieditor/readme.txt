SmartEditor Basic 2.0 설명

SmartEdtitor™는 Javascript로 구현된 웨브기반의 WYSIWYG 편집기입니다.
SmartEdtitor™는 WYSIWYG 방식 및 HTML 편집방식과 TEXT 방식을 제공하고 자유로운 서체 크기설정기능, 줄 간격설정기능, 단어 찾기/바꾸기 기능 등 편집에 필요한 다양한 기능을 제공하므로 사용자들은 SmartEdtitor™를 사용하여 쉽고 편리하게 원하는 형태의 글을 작성할수 있습니다.
또한 SmartEdtitor™의 구조는 기능을 쉽게 추가할수 있는 플러그인 구조로 되여 있어 정해진 규칙에 따라 플러그인을 만들기만 하면 됩니다.

SmartEdtitor™는 천지봉기술분소에서 React.js로 porting하였습니다.
지원하는 웨브열람기 환경은 다음과 같습니다.

* 지원하는 웨브열람기
Internet Explorer 7.0+ / 10.0-
FireFox 3.5+
Safari 4.0+
Chrome 4.0+

패키지에 포함된 화일은 아래와 같습니다.
/css : 편집기에서 사용하는 css화일
/img : 편집기에서 사용하는 그림화일
/js : 편집기를 적용할때 사용하는 JS 화일
/photo_uploader : 사진올리적재popup UI를 구성하는 화일
readme.txt : 간략한 설명
release_notes.txt : 릴리즈 노트
sample.php : SmartEditor2.html을 리용해 편집한 내용을 봉사기에서 받는 php 례제
smart_editor2_inputarea.html : 편집기의 편집령역을 나타내는 HTML로 편집기를 적용할 때 반드시 필요
smart_editor2_inputarea_ie8.html : smart_editor2_inputarea.html와 동일한 기능이나 사용자의 웨브열람기 Internet Explorer 8.x 이상인 경우에 사용
SmartEditor2.html : 편집기 견본페지. 편집기 적용시에도 참고 할수 있다.
SmartEditor2Skin.html : 편집기를 적용한 페지에서 적재하는 편집기의 스킨 HTML 화일로 편집기에서 사용하는 JS 화일과 css 화일을 초련결하며 편집기의 마크업을 가지고 있다. SmartEditor2.html 에서도 확인할수 있다.
src_include.txt : 자바스크립트 플러그인 원천코드를 직접 수정하고자 할 경우 참고할수 있는 화일
